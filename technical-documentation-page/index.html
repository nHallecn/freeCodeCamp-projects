Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM).
  • Server-side JavaScript extends the core language by supplying objects relevant to running on a server.
  • JavaScript is often used to add interactive effects to web pages.
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Good working knowledge of HyperText Markup Language (HTML).
  • Some programming experience, if you want to work on server-side applications.
  • Familiarity with CSS.
  • Basic understanding of client-side web technologies.

If you don't have these skills, you should go back and brush up on them.

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others. The JavaScript language resembles Java but does not have Java's static typing and strong type checking.

JavaScript's primary design goals were to provide a scripting language to complement Java's applications and to be a lightweight language for the Web.

To declare a variable in Java:

int myNumber = 5;

While in JavaScript, it's:

let myNumber = 5;
Hello World

To get started with writing JavaScript, open the Console and type the following simple line of code:

console.log("Hello World!");

This prints the message "Hello World!" to the web console. Go ahead and try it yourself.

The standard convention for an HTML script tag is:

<script> // JavaScript code goes here </script>
Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, an underscore (_), or a dollar sign ($). Subsequent characters can also be digits (0-9).

The three ways to declare a variable are `var`, `let`, and `const`:

  • Using `var`: var x = 42;
  • Using `let`: let y = 13;
  • Using `const`: const PI = 3.14;

It is best practice to use `let` and `const` over `var` in modern JavaScript.
